# Nintendo Financial Performance Analysis (2021–2025)

## Overview

This project analyzes Nintendo's financial performance between FY2021 and FY2025 using data from Nintendo's annual reports.

The analysis focuses on:
- Revenue trends
- Operating profit trends
- Cash reserves
- Nintendo Switch hardware sales
- Software sales
- Digital sales

## Key Findings

- Revenue declined 30.3% in FY2025.
- Operating profit declined 46.6%.
- Nintendo maintained a strong cash position despite lower sales.
- Consumer anticipation of Nintendo Switch 2 may have influenced purchasing behavior.
